# html-to-markdown
conver html content to markdown form url

## how to use
1. install  
`yarn install @0xdwong/html-to-markdown`

2. import module  
    ```
    const htmlToMarkdown = require('@0xdwong/html-to-markdown');
    const url = 'input url here';
    const mdContent = await htmlToMarkdown(url);
    ```
